# DEZIRE  
### Fashion in Every Detail
DEZIRE is a digital fashion showcase inspired by modern retail aesthetics.  
The project focuses on clean design, curated categories, and a minimalist
visual language that reflects contemporary fashion culture.
---
## About the Project
DEZIRE introduces a stylish and intuitive platform where users can explore
fashion collections in a clean and organized way.  
The project is designed to evolve into a full online store experience,
combining aesthetics with functionality.
---
## Database
Table: products  
Fields:
- id
- name
- price
- category
- image
- description

## API Endpoints
- GET /api/products
- GET /api/products/:id
- POST /api/products
- PUT /api/products/:id
- DELETE /api/products/:id

## Features
- Full CRUD operations
- Search products using query parameters
- Product page by ID
- Static image support
---
## Technologies Used
- Node.js
- Express.js
- HTML
- CSS3
- File System module (`fs`)
